# AttendEase 📊
A modern, web-based student attendance management dashboard developed as a final semester BCA major project by our team (including Harshita, Riya, and Mamta).

## Key Features
* **Teacher/Admin Dashboard:** Real-time metrics on class attendance averages and defaulters.
* **Subject-Wise Tracking:** Allows faculty to manage and record attendance for specific subjects.
* **Automated Status:** Automatically flags students as 'Regular', 'At Risk', or 'Defaulter' based on a 75% threshold.
* **Data Export:** Built-in CSV report generation for monthly academic reviews. 

## Tech Stack
* HTML5
* CSS3 (Custom UI/UX with responsive design)
* Vanilla JavaScript (Logic and state management)
